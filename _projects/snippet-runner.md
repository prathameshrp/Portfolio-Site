Snippet Runner turns static code samples into interactive experiments. Readers
can edit and execute JavaScript right where they're reading — no context switch,
no separate tab.

## Highlights

- Runs untrusted code safely inside a Web Worker sandbox.
- Captures `console` output and return values into an inline console.
- Tiny footprint, framework-agnostic, drops into any site.
