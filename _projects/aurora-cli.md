Aurora CLI takes the friction out of going from an empty folder to a deployed
site. It scaffolds a project, wires up CI/CD, and ships to GitHub Pages with a
single command.

## Why I built it

I kept rewriting the same deploy pipeline for every small project. Aurora bakes
the good defaults in so you never start from scratch again.

## Highlights

- Sub-second cold start written in Go.
- Pluggable templates for blogs, docs, and landing pages.
- Generates a ready-to-go GitHub Actions workflow.
